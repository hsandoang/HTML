Introduction to HTML

This is a simple introduction to HTML. HTML stands for HyperText Markup Language. It is the standard markup language for creating web pages.

In January 28, 1990, HTML was first proposed by Tim Berners-Lee. But, it wasn't until the early 1990s that HTML became widely adopted. The example of HTML in action.  Here is a list of some common HTML tags:

  • <h1> to <h6> - Headings
  • <p> - Paragraph
  • <a> - Anchor
  • <img> - Image

HTML is a powerful language for creating structured web content. because it allows developers to structure their content in a semantic and meaningful way. You can use these tags to create well-structured and accessible web pages Click here.
